# Manual Setup Steps

After the automated setup completes, the following manual steps are recommended:

## System

-   Log in with your Apple account / iCloud credentials in System Settings

## Raycast

-   Launch the app manually and complete the setup
-   Rebind `Cmd+Space` from Spotlight to Raycast

## 1Password

-   **Safari browser extension**: Install manually from the App Store
-   **SSH Agent**: Enable manually in the 1Password app:
    1. Go to Settings â†’ Developer
    2. Check "Show 1Password Developer Experience"
    3. Enable "SSH Agent - Use SSH Agent" (no need to configure anything since nix handles that)
    4. Optionally, set "Remember Key Approval" to a different value that's more optimal

## Xcode

-   Install Xcode through the App Store

## Cleanshot X

-   Run the application manually and enter the license key
-   Follow the setup in the app

## Cursor

-   Log in within the app

## Discord

-   Log in within the app

## Spotify

-   Log in within the app
